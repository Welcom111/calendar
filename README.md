# Calendar

Небольшое приложение в системном трее для отображения событий из CalDAV.

Настройки CalDAV изменяется в настройках по нажатию ⚙

# Запуск

Скачайте последнюю версию программы из [релизов](https://github.com/Welcom111/calendar/releases).

Наведите указатель на значок в трее, чтобы увидеть время с секундами и дату.

Нажмите на значок чтобы увидеть все события за выбранный месяц, по-умолчанию открывается текущий месяц.
