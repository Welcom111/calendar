# QuickCalendar

Значок в системном трее и календарь с событиями CalDAV.

## Запуск

1. Установите Python 3.10+ с опцией `Add Python to PATH`.
2. Один раз запустите `install.bat`.
3. Запускайте приложение через `run.bat`.

После обновления приложения также повторно запустите `install.bat`, чтобы
установились новые зависимости, включая парсер календарных событий `vobject`.

Наведите указатель на значок в трее, чтобы увидеть время с секундами и дату.
Нажмите на значок, затем откройте настройки кнопкой-шестерёнкой и укажите
URL CalDAV, имя пользователя и пароль приложения. Для Nextcloud обычно подходит
URL вида `https://cloud.example.com/remote.php/dav`. Если поле названия календаря
оставить пустым, события загружаются из всех доступных календарей.

При открытии показываются все события текущего месяца. Переключение месяца
стрелками календаря загружает события отображаемого месяца. Сетевые запросы
выполняются в фоне.

Для автономного EXE запустите `build-exe.bat`; результат появится в
`dist\QuickCalendar.exe`.
